Visual Data

Tables

All tables should be written symantically using tags in order to signify the table headers.

Use the scope attribute in order to associate the table header with a row or column.

Use the <caption> tag in order to write a small description of the table's purpose for disabled users. This text (if not in the design) can be hidden with the techniques in "Hidden Screen Reader Content".

Graphs/Charts

Text positioned near the chart and hidden offscreen should be supplied for any visual data on a page. This allows disabled users to understand the purpose of the chart/graph without seeing the data.

Ensure that the graph/chart's summary is short and succinct and only relay the purpose/results of the chart or graph and not the actual data points in their entirety.
