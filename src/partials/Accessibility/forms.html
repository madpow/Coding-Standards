Forms

All input fields should have a label associated with them for accessibility. If the project's design doesn't include a label, code one anyways and hide it using the techniques in "Hidden Screen Reader Content".

Use fieldset and legend for groups of inputs like radio buttons or checkboxes so that disabled users can see what selections are available within that grouping as well as the context of the selection.

Ensure that all input fields have a visible focus style. This could be an outline around the input or a background color change. As long as it is visually apparent to the user.
