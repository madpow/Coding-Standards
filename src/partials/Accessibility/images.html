Images

Image should always have an alt attribute, the following rules apply to image alt tags:

  • Write a brief description of the image in case it cannot be loaded or if the user is visually impaired and cannot see it.
  • If the image is not adding anything to the content (ex. A page divider, an icon), then the alt tag can be left blank.

    • When images don't have an alt tag associated with them at all a screen reader will speak the file name.
