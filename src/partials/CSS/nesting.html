Nesting

Do not nest styles unless you are scoping those styles to a parent. Although nesting may appear to keep code organized in SASS or LESS it causes the compiled CSS code to become unreadable. In addition, nesting can cause scoping issues throughout the project.

Consider creating styles in a more modular fashion in order to reduce the need for nesting.

If nesting is necessary, avoid going more than 3 levels deep.

// Avoid .selector div p span { // styles } .selector-two { div { p { span { // styles } } } } // Instead .selector { // styles } .div { // styles p { // styles } } span.selector { // styles }
