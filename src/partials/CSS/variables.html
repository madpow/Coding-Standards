Variables

Variables should be used for values that are going to be used globally throughout the project.

Generally, these declarations are made into global variables:

  • Colors
  • Typography
  • Media Query Breakpoints

Variable names should be written so they are clear as to what value they are holding.

Variables should be declared at the top of a individual stylesheet if pertaining only to that component.

$color-brand-primary: #fff; $breakpoint-sm: 768px; $font-primary: 'Helvetica Neue', Arial, sans-serif;
