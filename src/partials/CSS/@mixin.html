@mixin

Mixins should be used throughout the project for styles that are repetitive to keep CSS as DRY as possible.

// Example of a mixin for a button @mixin button($color) { background-color: $color; border: 1px solid mix(black, $color, 25%); border-radius: 5px; padding: .25em .5em; &:hover { cursor: pointer; background-color: mix(black, $color, 15%); border-color: mix(black, $color, 40%); } } .button { @include button(#b4d455); }
