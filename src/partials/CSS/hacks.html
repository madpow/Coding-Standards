Note: these rules should not be used unless there is no other option.

In order to increase specificity without nesting, class selectors can be chained with themselves.

For third party components where source code cannot be altered, the ID can be used as an attribute selector. This ensures the same amount of specificity as a class.

.selector.selector { background-color: #cfa338; } [id="camelCase"] { color: #fff; }
