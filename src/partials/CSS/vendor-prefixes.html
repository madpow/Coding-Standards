Vendor Prefixes

To ensure that elements appear the same way across all browsers, vendor prefixes should be utilized. When possible, incorporate the insertion of vendor-prefixed selectors in with your build process. Doing so will allow you to just write CSS as you normally would, without stopping to insert prefixes. Grunt/Gulp - Use of Autoprefixer is preferred:

  • Grunt: http://grunt-tasks.com/autoprefixer/
  • Gulp: https://www.npmjs.com/package/gulp-autoprefixer
.selector { -webkit-box-shadow: 7px 7px 5px 0px rgba(50, 50, 50, 0.75); -moz-box-shadow: 7px 7px 5px 0px rgba(50, 50, 50, 0.75); box-shadow: 7px 7px 5px 0px rgba(50, 50, 50, 0.75); } // Using Autoprefixer only require the following .selector { box-shadow: 7px 7px 5px 0px rgba(50, 50, 50, 0.75); }
