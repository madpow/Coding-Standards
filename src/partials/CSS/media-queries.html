Media Queries

When using media queries, place them close to their relevant rule sets.

Do not place media queries in separate stylesheets from their rule sets.

.selector { // styles } .selector-two { // styles ul { // styles } } @media only screen and (min-width: 768px) { .selector { // styles } .selector-two { // styles ul { // styles } } } @media only screen and (min-width: 940px) { .selector { // styles } .selector-two { // styles ul { // styles } } } // OR ? .selector { // styles @media only screen and (min-width: 768px) { // styles } @media only screen and (min-width: 940px) { // styles } } .selector-two { // styles ul { // styles } @media only screen and (min-width: 768px) { // styles ul { // styles } } @media only screen and (min-width: 940px) { // styles ul { // styles } } }
