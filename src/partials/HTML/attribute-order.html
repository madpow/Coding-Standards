Attribute Order

HTML attributes should be generally listed in this order for clarity and legibility:

  • ID
  • Class
  • Name
  • Src, href, value, for, type
  • Title, alt
  • Role, aria-*
  • Booleans

We put IDs first because they are unique identifiers that should be used sparingly and only as JS hooks. Therefore, it is important to see them immediately for understandability and debugging purposes.
