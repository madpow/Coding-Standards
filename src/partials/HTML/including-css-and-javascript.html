Including CSS and Javascript

Include all CSS files within the head and Javascript files before the closing body tag of your page.

There is no need to include the type attribute when linking CSS or Javascript files as the browser automatically defaults to their correct values.

    		

		    	<!DOCTYPE html>
					<html lang="en-us">
						<head>
							<meta charset="utf-8">
							<!-- External CSS Stylesheet -->
							<link rel="stylesheet" href="...">
						</head>
						<body>
							...
							<!-- Javascript -->
							<script src="main.js"></script>
						</body>
					</html>
